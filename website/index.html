Great for Polish users

Polish diacritics (ą ę ś ć ż ź ó ł ń) are typed with Right Option + letter. On many external keyboards, Right Option is where Command sits on the MacBook — this swap fixes that.

Installation

One command to download and activate the key swap. Works on any Mac.

Install via Terminal
$ curl -fsSL https://nikoro.github.io/swap-right-command-and-option/install.sh | bash
Downloads the LaunchAgent plist to ~/Library/LaunchAgents/ and loads it immediately.
Prefer to install manually?
1 Save the plist file to ~/Library/LaunchAgents/
2 Load the agent:
launchctl bootstrap gui/$(id -u) ~/Library/LaunchAgents/com.local.swap-right-command-and-option.plist
To uninstall, see uninstall instructions on GitHub.
